#P3_mastermind

Bonjour

Vous trouverez ici le travail que j'ai réalisé pour ce projet.

Pour lancer l'application, vous pouvez double-cliquer sur le fichier MasterMind.bat.
Pour lancer l'application en mode développement, double-cliquez sur le fichier MasterMind_developpement.bat.

Le dossier UML contient différents schémas UML que j'ai réalisé pour ce projet.
Le dossier Ressources contient les fichiers Image utilisés dans le programme et le fichier de configuration .properties.
Le dossier lib contient les librairies nécessaires au programme (ici apache-log4j2).

Le fichier .xml de configuration du logger se trouve dans le dossier src. Les logs sont écrites dans le fichier all.log.

Bonne lecture
